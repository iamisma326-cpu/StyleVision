# Plan: Fix asset paths for Vercel deployment

## Problem
Vercel serves HTML files via rewrites (e.g., `/` → `/html/index.html`). The browser sees URL as `/`, but HTML uses `../css/`, `../assets/`, `../js/` which resolve incorrectly from `/`.

## Solution
Change all `../` relative paths to absolute `/` paths in all HTML files.

## Files to modify
- `html/index.html` - 12 paths
- `html/como-funciona.html` - 4 paths
- `html/explorar.html` - 12 paths
- `html/contacto.html` - 2 paths
- `html/herramienta-ia.html` - 2 paths

## Changes
Replace in all files:
- `href="../css/` → `href="/css/`
- `src="../assets/` → `src="/assets/`
- `src="../js/` → `src="/js/`
