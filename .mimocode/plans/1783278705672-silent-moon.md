# Plan: Vercel outputDirectory configuration

## Goal
Move `index.html` back to `html/` and configure Vercel to serve from `html/` directory.

## Steps

1. **Move `index.html` back to `html/`**
   ```
   git mv index.html html/index.html
   ```

2. **Create `vercel.json`** at project root:
   ```json
   {
     "outputDirectory": "html"
   }
   ```

3. **Commit and push**
   ```
   git add -A && git commit -m "Configure Vercel outputDirectory to html/" && git push
   ```

## Verification
- After push, Vercel should detect the config and serve from `html/`
- Check the Vercel deployment URL works
